Raw SVG Code

Save this code as favicon.svg. Modern browsers support SVG favicons directly, which ensures they look incredibly crisp on high-resolution displays. 

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <defs>
    <linearGradient id="signalGrad" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%" stop-color="#818cf8" />
      <stop offset="100%" stop-color="#4F46E5" />
    </linearGradient>
  </defs>
  <!-- Core Dot -->
  <circle cx="50" cy="50" r="20" fill="url(#signalGrad)"/>
  
  <!-- Broadcast Ring -->
  <circle cx="50" cy="50" r="38" fill="none" stroke="url(#signalGrad)" stroke-width="10"/>
</svg>

To use this in your HTML, add this to your <head> tag:
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
